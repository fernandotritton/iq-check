# Cómo obtener tu Supabase Anon Key

1. Ve a tu dashboard de Supabase.
2. Click en el ícono de engranaje **(Project Settings)** en la barra lateral izquierda.
3. Selecciona **"API"** en el menú.
4. En la sección **"Project API keys"**, verás dos claves: `anon` y `service_role`.
5. Copia la clave **`anon`** (public).
6. Pégala en el chat.

Esta clave nos permitirá conectar tu aplicación a la base de datos usando HTTPS, evitando los problemas de bloqueo de puertos que estamos experimentando.
